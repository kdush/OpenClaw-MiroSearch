# 行为准则

> 📄 English version: [CODE_OF_CONDUCT_en.md](./CODE_OF_CONDUCT_en.md)

本项目致力于构建开放、专业、尊重他人的协作环境。

## 我们期望的行为

- 讨论问题聚焦技术事实与可验证证据
- 尊重不同背景、经验与观点
- 在评审与讨论中给出可执行建议
- 对错误及时更正并保持透明沟通

## 不可接受的行为

- 人身攻击、侮辱性语言、骚扰行为
- 恶意刷屏、挑衅、歧视性言论
- 未经允许公开他人隐私信息
- 明知有风险仍传播敏感漏洞细节

## 执行与处理

维护者有权对违反行为准则的内容进行编辑、删除或限制互动。
对严重或持续违规行为，维护者可暂停或取消其参与权限。
