# Code of Conduct

> 📄 中文版：[CODE_OF_CONDUCT.md](./CODE_OF_CONDUCT.md)

This project is committed to building an open, professional, and respectful collaboration environment.

## Expected Behavior

- Focus discussions on technical facts and verifiable evidence
- Respect different backgrounds, experiences, and perspectives
- Provide actionable suggestions during reviews and discussions
- Correct errors promptly and maintain transparent communication

## Unacceptable Behavior

- Personal attacks, insulting language, or harassment
- Malicious spamming, provocation, or discriminatory speech
- Publishing others' private information without permission
- Deliberately spreading sensitive vulnerability details knowing the risk

## Enforcement

Maintainers have the right to edit, remove, or restrict interaction on content that violates this code of conduct.
For severe or persistent violations, maintainers may suspend or revoke participation privileges.
