# 支持说明

## 适用范围

本仓库支持以下场景：

- OpenClaw-MiroSearch 部署与配置问题
- API 调用与参数选择问题
- 检索质量、超时、卡住等运行问题

## 获取支持

请优先通过 GitHub Issue 提交问题，并尽量附带以下信息：

- 使用分支与提交号
- 运行方式（Demo / API / Agent）
- 关键环境变量（请脱敏）
- 复现步骤与预期结果
- 报错日志或截图

Issue 模板：

- Bug：`.github/ISSUE_TEMPLATE/bug_report.md`
- Feature：`.github/ISSUE_TEMPLATE/feature_request.md`

## 响应边界

- 开源版本以社区协作为主，不提供 SLA 承诺
- 涉及密钥泄露、账号风险等安全问题，请按 [`SECURITY.md`](SECURITY.md) 私下报告
